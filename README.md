# Expert System — PC Diagnostics

A rule-based expert system that diagnoses PC hardware failures 
from user-reported symptoms using a forward chaining engine.

## Features
- Symptom selection via CLI
- Forward chaining inference engine
- Rule-based reasoning (7 production rules)

## Usage
```bash
python diagnostic.py
```

## Tech
Python 3 — no external libraries
